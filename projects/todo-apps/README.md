# Django Todo APP

A simple Todo Management System built with Django.

## Including Some Features:
- Create Tasks
- Task Priorities (High, Medium, Low)
- Categories
- Due Dates
- Overdue Detection
- Completion Tracking
- Search Tasks
- Filter Tasks
- Admin Dashboard

## To Priorites the Task, include Sroting Logic:

1. Incomplete tasks first
2. High → Medium → Low priority
3. Earliest due date first
